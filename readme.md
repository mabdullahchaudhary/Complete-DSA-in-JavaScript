## 🎯 About the Course

This course is a comprehensive deep dive into computer science fundamentals. It focuses on writing efficient code and understanding the underlying mechanics of data structures and algorithms using **TypeScript**.

---

## 🚀 How to Run the Code

Since this project uses TypeScript, I am using `tsx` to execute the files directly without the need for manual compilation.

### 1. Requirements

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Setup

To run the files without typing `npx` every time, install `tsx` globally:

```bash
npm install -g tsx
```

### 3. Execution

You can run any file from the project root using:

```bash
tsx folder_name/file_name.ts
```
